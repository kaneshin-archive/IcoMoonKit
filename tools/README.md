# Tools

## Requirement

- Go
- Ruby

## Installation

### IcoMoon Generator

```
$ go get github.com/kaneshin/IcoMoonKit/tools/icom
```


## Usage

### Go

```
$ icom selection.json
```

### Ruby

```
$ ./icomoon.rb selection.json
```
